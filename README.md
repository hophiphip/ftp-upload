# FTP Upload

FTP Upload helper for `Powershell`

## Quick start
```sh
./FTPUpload.ps1
```

## Use module
Clone this repository
```sh
git clone https://github.com/hophiphip/ftp-upload
```

```sh
cd ftp-upload
```

Import module
```powershell
Import-Module FTPUload
```
